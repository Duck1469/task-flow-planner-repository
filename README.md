# TaskFlow Planner

A responsive 3-page to-do planner with:
- **Today page**: add, search, complete, and schedule tasks by hour.
- **Calendar page**: monthly completion heatmap.
- **Settings page**: theme, text size, and completion colors.

## Run locally

```bash
python3 -m http.server 4173
```

Then open: <http://localhost:4173>

## Notes
- Data is automatically saved in browser `localStorage`.
- For cross-device sync, use **Export data** on one device and **Import data** on another.
